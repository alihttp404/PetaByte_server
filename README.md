# PetaByte Server

This is a backend CRUD API made to handle database requests to PostgreSQL for PetaByte mobile app, written in node js and express. The server is hosted on Render, as well as the database.


The main architecture of the project can be summarized as the following:

Development:
Frontend: Flutter
Backend: Express (node js)
Database: Postgresql
Authentication: Firebase
Hostinng service: Render

Version Control:
dev - used for development
main - used for production(render)

Testing:
Render - a pr testing server is used before deployment
Postman - CRUD API testing
