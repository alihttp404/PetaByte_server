# PetaByte Server

This is a backend CRUD APII made to handle database requests to PostgreSQL for PetaByte mobile app, written in node js and express. The server is hosted on Render, as well as the database.


The main architecture of the project can be summarized as the following:

## Development:

  Frontend: Flutter
  
  Backend: Express (node js) 
  
  Database: PostgreSQL
  
  Authentication: Firebase
  
  Hosting service: Render


## Version Control:

  Dev - used for development and testing
  
  Main - used for production(render)


Testing:

  Render - Once a PR is created, a stage area is automatically generated
  
  Postman - CRUD API testing
